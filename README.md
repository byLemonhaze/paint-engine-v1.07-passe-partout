## What It Is

Paint Engine v1.07 is both a working tool and a permanent artifact. The engine lives entirely in a single HTML file — no server, no dependencies, no network calls — and runs in any modern browser. It generates layered, painterly compositions through a deterministic seed + tag system, producing outputs that feel hand-made rather than algorithmic.

The *Passe-Partout* edition is a self-contained version of the engine designed to stand alone as a collectible inscription. It represents the state of the tool at a specific moment in the ongoing paint engine lineage.

---

## The Output

Each session produces a unique layered canvas painting. The visual character is determined by a master seed combined with a set of tags — palette, brush type, background treatment, texture overlay, and frame. Given the same seed and tags, the engine reproduces the exact same image. This means collectors can save, share, and reconstruct any output precisely.

---

## Seed & Tag System

| Parameter | What It Controls |
|-----------|-----------------|
| **Master Seed** | The root of the composition — determines placement, structure, and variation |
| **Palette** | Color family and tone — drawn from a curated set of named palettes |
| **Iteration** | Sub-variation within a seed — iterate without losing context |
| **Background** | Ground layer treatment |
| **Brush** | Mark-making style and weight |
| **Texture** | Surface overlay — grain, noise, layering |
| **Frame** | Presentation edge — varies from clean crop to compositional border |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `G` | Generate — new seed, fresh composition |
| `I` | Iterate — variation while holding seed + palette context |
| `B` | Back — step through generation history |
| `W` | Parameters — toggle seed/tag display and edit |
| `C` | Customize — open palette, ratio, brush, texture, frame controls |
| `S` | Save — export current canvas as PNG |
| `#` | Grid View — browse multiple outputs side by side |
| `L` | Collection — open favorites/saved outputs |
