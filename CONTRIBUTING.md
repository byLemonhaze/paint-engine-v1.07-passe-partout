# Contributing

Thanks for contributing to Paint Engine v1.07 [Passe-Partout].

## Workflow

1. Create a branch from main.
2. Make focused changes.
3. Run verification locally.
4. Open a PR with a clear summary.

## Required Checks

- npm run verify

## Standards

- Keep generated HTML deterministic when expected.
- Preserve inscription-compatible output behavior.
- Do not commit secrets.
